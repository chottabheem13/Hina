# Retur Monitor

- Command: `/warehouse-ticket`
- Kategori: `Pindah Fisik`
- Ticket value: `return_monitor`
- Channel env: `CHANNEL_PINDAHFISIK`
- Thread prefix: `RETMON`

## Field

- Wajib: `order_id`, `note`, `assigned_to`
- Opsional: tidak ada

## Cara Buat

1. Jalankan `/warehouse-ticket`
2. Pilih `Pindah Fisik`
3. Pilih `Retur Monitor`
4. Isi `Order ID`
5. Isi `Note`
6. Pilih minimal 1 assignee lalu submit

## Catatan

- `note` dipakai untuk catatan atau keterangan retur
