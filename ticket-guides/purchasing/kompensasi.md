# Kompensasi

- Command: `/purchasing-ticket`
- Ticket value: `kompen`
- Channel env: `CHANNEL_KOMPEN`
- Thread prefix: `KOMPEN`

## Field

- Wajib: `order_id`, `notes` sebagai description, `assigned_to`
- Opsional: `priority`

## Cara Buat

1. Jalankan `/purchasing-ticket`
2. Pilih `Kompensasi`
3. Isi `Order ID`
4. Jelaskan defect atau damage di field `Description`
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- `priority` default ke `Normal`
- Embed otomatis membuat link order dari `order_id`
