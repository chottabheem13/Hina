# ETA (UREQ)

- Command: `/purchasing-ticket`
- Ticket value: `eta_ureq`
- Channel env: `CHANNEL_UREQ`
- Thread prefix: `UREQ`

## Field

- Wajib: `order_id`, `assigned_to`
- Opsional: `priority`, `notes`

## Cara Buat

1. Jalankan `/purchasing-ticket`
2. Pilih `ETA (UREQ)`
3. Isi `Order ID`
4. Pilih minimal 1 assignee
5. Submit modal

## Catatan

- `priority` default ke `Normal`
- Embed otomatis membuat link order dari `order_id`
