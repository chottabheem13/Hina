# New Item (Pre-order)

- Command: `/purchasing-ticket`
- Ticket value: `new_item_preorder`
- Channel env: `CHANNEL_PPO`
- Thread prefix: `NEWDB`

## Field

- Wajib: `notes` sebagai item description, `assigned_to`
- Opsional: `link`

## Cara Buat

1. Jalankan `/purchasing-ticket`
2. Pilih `New Item (Pre-order)`
3. Isi deskripsi item di field `Item Description`
4. Tambahkan `Link` bila ada referensi
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Field deskripsi disimpan di `notes`
- Cocok untuk request kompre item baru
