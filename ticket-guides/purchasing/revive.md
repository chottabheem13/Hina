# Revive

- Command: `/purchasing-ticket`
- Ticket value: `revive`
- Channel env: `CHANNEL_REVIVE`
- Thread prefix: `REVIVE`

## Field

- Wajib: `item_id`, `assigned_to`
- Opsional: `notes`

## Cara Buat

1. Jalankan `/purchasing-ticket`
2. Pilih `Revive`
3. Isi `Item ID`
4. Tambahkan catatan bila perlu
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Embed otomatis membuat link item dari `item_id`
