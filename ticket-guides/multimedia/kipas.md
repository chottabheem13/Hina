# Kipas

- Command: `/mulmed-ticket`
- Kategori: `Offset Printing`
- Ticket value: `kipas`
- Channel env: `CHANNEL_OFFSET`
- Thread prefix: `KIPAS`

## Field

- Wajib: `brief`, `assigned_to`
- Opsional: `priority`, `link`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Offset Printing`
3. Pilih `Kipas`
4. Isi `Brief` dengan size dan quantity
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Brief sebaiknya memuat size dan QTY
