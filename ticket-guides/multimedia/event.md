# Event

- Command: `/mulmed-ticket`
- Kategori: `Event Design`
- Ticket value: `event`
- Channel env: `CHANNEL_EVENT`
- Thread prefix: `EVENT`

## Field

- Wajib: `brief`, `assigned_to`
- Opsional: `priority`, `link`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Event Design`
3. Pilih `Event`
4. Isi `Brief`
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Brief sebaiknya memuat deadline, scope, dan info tambahan
