# Kolase

- Command: `/mulmed-ticket`
- Kategori: `Digital Design`
- Ticket value: `kolase`
- Channel env: `CHANNEL_KOLASE`
- Thread prefix: `KOLASE`

## Field

- Wajib: `assigned_to`
- Opsional: `priority`, `link`, `brief`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Digital Design`
3. Pilih `Kolase`
4. Isi brief bila diperlukan
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Bisa menampilkan specialist note
- Ada assignment user spesifik di `TICKET_USERS`
