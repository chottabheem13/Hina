# Other Printing

- Command: `/mulmed-ticket`
- Kategori: `Single Printing`
- Ticket value: `other_print`
- Channel env: `CHANNEL_SINGLEPRINT`
- Thread prefix: `OTHER-PRINT`

## Field

- Wajib: `assigned_to`
- Opsional: `priority`, `link`, `brief`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Single Printing`
3. Pilih `Other`
4. Isi kebutuhan printing di `Brief`
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Cocok untuk request single printing di luar subtype utama
