# Monthly Design

- Command: `/mulmed-ticket`
- Kategori: `Digital Design`
- Ticket value: `monthly_design`
- Channel env: `CHANNEL_DIGITAL`
- Thread prefix: `MONTHLY`

## Field

- Wajib: `assigned_to`
- Opsional: `priority`, `link`, `brief`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Digital Design`
3. Pilih `Monthly Design`
4. Isi brief dan output tambahan bila perlu
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Ada assignment user spesifik di `TICKET_USERS`
