# Project

- Command: `/mulmed-ticket`
- Kategori: `Event Design`
- Ticket value: `project`
- Channel env: `CHANNEL_EVENT`
- Thread prefix: `PROJECT`

## Field

- Wajib: `assigned_to`
- Opsional: `priority`, `link`, `brief`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Event Design`
3. Pilih `Project`
4. Isi `Brief` bila ada requirement khusus
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Brief opsional, tetapi sebaiknya tetap isi scope dan deadline
