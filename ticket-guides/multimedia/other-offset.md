# Other Offset

- Command: `/mulmed-ticket`
- Kategori: `Offset Printing`
- Ticket value: `other_offset`
- Channel env: `CHANNEL_OFFSET`
- Thread prefix: `OTHER-OFFSET`

## Field

- Wajib: `assigned_to`
- Opsional: `priority`, `link`, `brief`

## Cara Buat

1. Jalankan `/mulmed-ticket`
2. Pilih `Offset Printing`
3. Pilih `Other`
4. Isi `Brief` jika ada kebutuhan size, QTY, atau output
5. Pilih minimal 1 assignee
6. Submit modal

## Catatan

- Cocok untuk request offset di luar subtype utama
